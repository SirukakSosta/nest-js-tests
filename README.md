# Nest JS tests

NestJS
MongoDB

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Run Docker mongo db
$ docker-compose up
```

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

# Mongo Migrations

```bash
$ npm install -g migrate-mongo
```

see : https://www.npmjs.com/package/migrate-mongo
